# bGeigieCast

An extension module to convert your mobile bGeigieNano to a fixed sensor, like a pointcast! 

## Hardware requirements

* bGeigieNano
* bGeigieCast module
* Wireless charging
    * Dock
    * Transmitter
    * Receiver
